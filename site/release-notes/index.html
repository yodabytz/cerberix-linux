0.1.0 — Styx

Released 2026-04-18 x86_64 ISO: 3.0 GB

The first public Cerberix Linux release.

Core

  • Arch Linux rolling base with linux and linux-hardened kernels
  • XFCE 4.20 desktop with Tokyo Night Moon theming, Plank dock, and preloaded user dotfiles in /etc/skel/
  • fish default shell with Tide prompt, Fisher plugins (fzf.fish, done), and zoxide

Security stack (preconfigured)

  • nftables firewall, fail2ban, ufw/gufw
  • rkhunter + lynis + apparmor + arch-audit
  • keepassxc + gnome-keyring for credentials
  • bleachbit for privacy cleanup

Privacy & VPN

  • tor, torbrowser-launcher
  • wireguard-tools, openvpn, openconnect
  • proxychains-ng

Offensive / audit toolkit

  • hydra, hashcat, nikto, radare2, sqlmap

Modern CLI

  • starship, tmux, eza, bat, fd, fzf, ripgrep, lazygit, neovim, plocate

Cerberix-native tools

  • Cerberix Shield — system tray indicator for security posture
  • Cerberix Connect — firewall appliance dashboard
  • Cerberix Update / Cerberix rkhunter — systemd timers

Installer

  • Single-command: sudo cerberix-install
  • UEFI + BIOS, GPT + MBR
  • New users automatically inherit themed dotfiles via /etc/skel/

Known caveats

  • English locale only at install
  • Desktop only — no server variant yet
  • aide and hexchat ship as post-install AUR additions (yay -S aide hexchat)
  • 0.1.0 is young — please file issues generously

Download & verify

ISO + signature + checksum on the homepage. The signing key is published at /gpg.asc.
