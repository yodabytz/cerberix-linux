Krellix

Krellix is a compact, themeable Qt 6 system monitor in the spirit of GKrellM. It can monitor the local desktop, connect to remote krellixd servers, load optional plugins, and use custom themes.

Qt 6 remote daemon themes plugins pacman packaged deb rpm macOS port

First time

Add the signed cerberix-extra repo before installing packages.

repo setup

Install

Main client and built-in monitor set.

pacman -S krellix

Remote hosts

Install the standalone daemon on headless systems.

pacman -S krellix-server

Self-contained Intel macOS app bundle with bundled Qt runtime.
