Krellix plugins

Optional panels that can be installed only when wanted. The main Krellix package stays focused, and plugins can evolve independently.

Krellkam

Displays up to five periodically refreshed image/webcam panels, including supported live stream thumbnails.

pacman -S krellix-krellkam

Krelldacious

Audacious playback and volume controls in a compact Krellix panel.

pacman -S krellix-krelldacious

Krellweather

METAR weather conditions, temperature, and compact weather icon display using station IDs.

pacman -S krellix-krellweather

Krellwire

RSS headline ticker with click-through links to the original stories.

pacman -S krellix-krellwire

KrellSpectrum

Audio spectrum and waveform visualizer with multiple rendering modes.

pacman -S krellix-krellspectrum

Krellmail

POP3/IMAP unread mail count monitor with OAuth support for Gmail-style accounts.

pacman -S krellix-krellmail

KrellHealth

Compact local and remote health summary with kernel, journal, service, and package checks.

pacman -S krellix-krellhealth

KrellTop

Top CPU and memory process panel with remote krellixd process telemetry.

pacman -S krellix-krelltop

KrellStock

Stock quote ticker and detail panel for configured Yahoo Finance symbols, crypto pairs, and global market tickers.

pacman -S krellix-krellstock

KrellMoon

Moon phase panel with bundled phase artwork and hemisphere setting.

pacman -S krellix-krellmoon

KrellPkg

Package update monitor for configured package systems.

pacman -S krellix-krellpkg

KrellWiFi

WiFi status and signal panel for wireless systems.

pacman -S krellix-krellwifi

KrellStack

Container stack monitor for local and remote hosts.

pacman -S krellix-krellstack
