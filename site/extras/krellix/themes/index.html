Krellix themes

Bundled Krellix themes are available as standalone archives here. Click a screenshot thumbnail to see the full theme preview.

User install

Unpack a theme archive into your local Krellix theme directory.

~/.config/krellix/themes

Package install

The main package includes the bundled theme set.

pacman -S krellix
