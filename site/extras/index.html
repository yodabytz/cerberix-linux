cerberix-extra

Extra apps for Cerberix.

Tools and desktop utilities that ship through the signed cerberix-extra pacman repository. Krellix has its own page because it has installable plugins and downloadable themes.

Install the cerberix-extra repo

Add the signing key once, add the repo to /etc/pacman.conf, then install packages normally with pacman.

sudo pacman-key --init
curl -s https://cerberix.org/gpg.asc | sudo pacman-key --add -
sudo pacman-key --lsign-key 49840314FFF3DFE2D6E75439577040DEDD8E521E
[cerberix-extra]
SigLevel = Required DatabaseOptional
Server = https://repo.cerberix.org/$arch/
Put the repo block at the end of /etc/pacman.conf, then run sudo pacman -Sy. After that, package commands like sudo pacman -S krellix work directly.

Network

Netscope

Network visibility tool for watching active connections and system network behavior from the terminal.

pacman -S netscope
Memory

Swapwatch

Swap and memory pressure watcher for keeping an eye on systems before they get sluggish.

pacman -S swapwatch

Modsentry

Monitors ModSecurity logs and displays them in a readable format.

pacman -S modsentry

Fuzzytail

Fast colorized log tailing with filtering and themes, packaged as ft.

Fail2ban helper for reporting abusive hosts and connection details from the command line.
